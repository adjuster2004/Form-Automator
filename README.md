# Form Automator
Плагин для chrome (но должен работать и на других сайта) запоминает действия и повторяет сколько раз, сколько укажите.

## Плагин удобного управления ресурсами в OpenNebula

- Метки по задачам
- Удобный переход из Лайки в задачу Jira
- Имя и фамилия владельца стенда
- Изменяемая ширина левого меню
- Сортировка меток VM по алфавиту
- Контроль меток при создании новой (выводит напоминание, что метка уже существует)
- Автообновление списка VM
- Полная пагинация. Добавляены 1000 и 1500 VM на страницу
- Статистика по VM, экспорт в CSV

## 🛠️ Последовательность
- **Склонировать репозиторий**
```bash
git clone https://github.com/adjuster2004/Form-Automator
```

- **Добавить плагин в браузере**

Требуется включить режим разработчика

```
chrome://extensions/
```


- **На сайте появится кнопка управления**
