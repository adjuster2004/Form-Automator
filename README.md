# Form Automator
Плагин для chrome (но должен работать и на других сайта) запоминает действия и повторяет сколько раз, сколько укажите.

## Действия

- Нажимаем кнопку Записать действияя
- Проводим нужные действия на сайте
- Останавливаем запись
- Выставляем цифру, сколько раз нужно повторить записанные действия
- Жмем кнопку Повторить

## 🛠️ Последовательность
- **Склонировать репозиторий**
```bash
git clone https://github.com/adjuster2004/Form-Automator
```

- **Добавить плагин в браузере**

Требуется включить режим разработчика

```
chrome://extensions/
```


- **На сайте появится кнопка управления**
